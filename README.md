# POLAR Trial Genomic Analysis

This repository contains genomic and correlative analyses for the POLAR trial (Nature Medicine 2026).

## Repository structure
- data/ : processed datasets used in the manuscript figures and tables
- scripts/ : R scripts to reproduce analyses and figures
- figures/ : output figures generated by scripts

## How to run
1. Clone this repository
2. Open R / RStudio in the repository root
3. Run scripts in `scripts/` (each script is labeled by figure/table)

## Software
Analyses were performed in R (v4.x). Key packages include ggplot2, dplyr, maftools, survival, survminer, ggpubr, cowplot.

## Data access
Raw sequencing data are protected and available under controlled access subject to institutional review and data transfer agreement.
Processed analysis datasets and annotated mutation files used in this study are provided here and via Figshare upon publication.
